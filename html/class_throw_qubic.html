Main function controlling wizard qubic throwing.

Every call of function check avaliability of use this function (whether the wizard skills ale possible to use).

If use is possible, checks inputed key seuquences, if first key of sequence is down shows possible qubics positions and movment o player is blocked.Positions are set by player position.

If second key is inputed qubic is placed. Posobilities are deleted. Now player can move again.
